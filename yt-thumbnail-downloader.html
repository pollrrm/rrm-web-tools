YT Thumbnail Downloader

Paste YouTube URL + video title → downloads {Title}-thumb.jpg at the highest quality available. Built for the RRM Video Posting SOP.

Tip: Install the RRM WP Helper Chrome extension to enable the Send to RRM@home button on each card. See extension folder in the repo.

Filename will be {Title}-thumb.jpg with characters cleaned for Windows.

Detects every Video Title: + Video Link: pair in order. Works with the standard RRM task email format.
