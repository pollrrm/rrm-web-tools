RRM Web Support Tools

Browser-based utilities for routine website tasks. Single-file, no install, runs entirely in your browser.

YT Thumbnail Downloader

Paste the video posting task email — extracts YT IDs, dates, titles, content, and downloads thumbnails (named per SOP) for every post, grouped by industry.

Use for: RRM Video Posting (weekly, 3-4 posts per niche site)

DOCX Batch → WordPress

Drop a site's monthly ZIP — parses every .docx into WordPress-ready HTML, pairs each with its featured image, runs grammar/format/consistency checks.

Use for: Monthly blog batch posting per site

PDF Paste Formatter

Strips the hard line breaks PDFs leave behind on copy-paste. Joins wrapped lines, keeps real paragraph breaks, mends hyphenated words.

Use for: Cleaning up testimonials, bios, or any text copied from a PDF before posting

PDF → JPG

Convert PDF pages to JPG. Pick specific pages or page ranges, draw a crop box, cap max width, and tune JPG quality. Download individually or as a ZIP.

Use for: Pulling a single page out of a PDF as an image, or batch-exporting flyers/brochures for web use

RRM Newsletter → WPBakery

Parse the Unlayer HTML for layout (rows, columns, colors), then replace the text from the Word doc so the final copy and updated links go in. Outputs WPBakery shortcodes for the WordPress Code editor.

Use for: Posting the monthly RRM Elevate newsletter to ringringmarketing.com

SCMM Newsletter → WPBakery

Parse the HTML for layout, then replace text from the Word doc (final copy + updated links). Tuned for SCMM's Home Care Post format — combined heading+body blocks, multi-article columns separated by &nbsp;, el_id="w" and el_class="text-white" row markers.

Use for: Posting the Home Care Post Bulletin to seniorcaremarketingmax.com

DOCX Link Extractor

Drop a Word .docx to list every hyperlink — linked words + actual URL, with per-row copy buttons. Then paste your existing WordPress source to cross-check and auto-update out-of-date links (one at a time or all at once).

Use for: Auditing links in a coach-supplied doc, and updating the live links in a WP post to match

Content Compare

Compare a source file (PDF, DOCX, HTML) against a live URL. Side-by-side word-level diff flags missing or extra content so you can verify the page renders what the source says.

Use for: QA after pasting coach-supplied copy into a page — confirm nothing was dropped, auto-corrected, or duplicated
