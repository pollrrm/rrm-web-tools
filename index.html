RRM Web Support Tools

Browser-based utilities for routine website tasks. Single-file, no install, runs entirely in your browser.

YT Thumbnail Downloader

Paste the video posting task email — extracts YT IDs, dates, titles, content, and downloads thumbnails (named per SOP) for every post, grouped by industry.

Use for: RRM Video Posting (weekly, 3-4 posts per niche site)

DOCX Batch → WordPress

Drop a site's monthly ZIP — parses every .docx into WordPress-ready HTML, pairs each with its featured image, runs grammar/format/consistency checks.

Use for: Monthly blog batch posting per site

PDF Paste Formatter

Strips the hard line breaks PDFs leave behind on copy-paste. Joins wrapped lines, keeps real paragraph breaks, mends hyphenated words.

Use for: Cleaning up testimonials, bios, or any text copied from a PDF before posting
